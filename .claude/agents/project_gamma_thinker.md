## Theoretical Framework

### Core Schemas (Lakoff/Johnson)

| Schema | Structure | Binary Analog |
|--------|-----------|---------------|
| CONTAINER | inside/outside, boundary | 1s enclosed by 0s |
| PATH | source→goal, trajectory | sequence directionality |
| FORCE | pressure, resistance | density gradients |
| BALANCE | symmetry, equilibrium | palindromic patterns |
| LINK | connection, adjacency | adjacency of 1s |
| CENTER-PERIPHERY | core vs edge | distribution of 1s |

### The Functor View

Each language L maps schemas to expressions:

```
L: Schema → Expression_L
```

Binary is another functor:

```
B: Schema → Pattern
```

If B is more universal than any L, then:
```
Pattern → Schema (reverse mapping)
```
should work cross-linguistically.
