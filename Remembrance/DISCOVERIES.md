## D1: f_dot Reveals Non-Random Structure

**Date:** 2024-12-21

**Hypothesis:** H0 (existence of structured encoding) + H1 (f_dot)

**Claim:**
```
f_dot(Quran) produces a bitstring with statistically significant
non-random structure, as measured by compression ratio.
```

**Evidence:**
```
Full Quran encoding:
  - Bitstring length: 311,903 bits
  - Density (1s): 34.19%
  - Compression ratio: 0.1545

Null comparison (50 shuffles):
  - Z-score for compression: -17.59
  - p < 0.0001

Interpretation: The dot-pattern in the Quran text is significantly
MORE compressible than would be expected from random letter arrangement.
This indicates structured patterning of dotted vs undotted letters.
```

**Validation Level:** L2 (tested, awaiting replication)

**Falsification Test:**
```
If shuffled Quran text produces similar compression ratios,
this finding would be falsified. It does not.
```

**Implications:**
1. H0 is supported: at least one f reveals structure
2. The structure exists at the letter level (not just word/verse)
3. Void (undotted) runs are longer than existence (dotted) runs
4. The "message" (if any) favors extended void passages punctuated by existence

**Open Questions:**
- What causes this structure? Linguistic? Deliberate? Emergent?
- Does the structure correlate with semantic content?
- Can the pattern be interpreted meaningfully?
